# Insightly

A mobile-first analytics viewer for small business owners.

Built for people who don't need a dashboard — they need one graph and a plain explanation, on their phone.

## Live App
https://insightly-dun.vercel.app

## What It Does

**Analyst mode**
- Load any CSV file
- App auto-detects columns — text becomes dimensions, numbers become measures
- Add graphs, pick dimension and measure, write plain-English insights
- Export everything as one `.insightly` file

**Client mode**
- Load the `.insightly` file
- See one clean graph at a time
- Swipe to navigate between graphs
- Tap insight button to read the explanation
- No controls, no configuration, no confusion

## Tech Stack
- React + Vite
- Capacitor (Android APK)
- PWA (installable on any phone)
- Deployed on Vercel
- Custom `.insightly` file format
